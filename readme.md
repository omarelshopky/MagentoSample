# CREATE MAGENTO APP Sample project

## Installation

1. Clone repository

    ```bash
    git clone git@bitbucket.org:scandiweb/cma-sample-project.git
    ```

2. Install dependencies

    ```bash
    npm ci
    ```

3. Run CMA

    ```bash
    npm start
    ```

[Documentation](https://docs.create-magento-app.com/getting-started/getting-started)
